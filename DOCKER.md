# Docker Setup

## Quick Start

### 1. Create `.env` file

```bash
NEXT_PUBLIC_SUPABASE_URL=your_supabase_project_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
```

### 2. Run

```bash
# Start
docker-compose up --build

# Stop
docker-compose down
```

## That's it!

Your app will be available at `http://localhost:3000`
